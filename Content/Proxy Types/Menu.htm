Menu Proxy

The Menu Proxy is used to generate a context-sensitive navigation menus for HTML output. They can generate two types of menus:

  • TOC Menus: A TOC Menu is a navigation element that shows a topic's parent, siblings, and children topics.

  • Heading Menus: A Heading Menu shows the headings used in a topic.

This topic has two menu proxies. The one on the left is a TOC Menu, and it is included in the project MasterPage. The one on the right is a Heading Menu. It is included in the topic itself.

How the Menu Proxy works in this project

A Menu Proxy is included in the default.flmsp MasterPage. This proxy is used as a TOC Menu, and it is included on every page in the HTML output. The default.flmsp MasterPage is assigned to the HTML5 target (see the Advanced tab of the target). When you build the HTML5 target, the default.flsmp MasterPage is applied to each topic, and the Flare TOC is built into a context-sensitive menu that replaces the Menu Proxy on each page in the HTML output.Learn more at http://help.madcapsoftware.com/flare2018/Content/Flare/Proxies/Creating-Topic-Menu.htm

In most projects, the menu proxy and body proxy are placed side-by-side. This requires some styles in the stylesheet to make this happen. In this project, the menu and body proxies are placed inside a responsive row that was created using Flare's Responsive Layout editor.Learn more at http://help.madcapsoftware.com/flare2018/Content/Flare/Interface/UI-Elements/Responsive-Layout-Window-Pane.htm

In addition to the Menu Proxy in the MasterPage, this topic also includes a Menu Proxy that generates just this topic's headings.

List-Of Proxy Dependencies

For TOC Menus: A TOC file with a Menu Proxy in a MasterPage or Topic.

For Heading Menus: A topic with multiple headings and a Menu Proxy it in. In the Menu Proxy's properties, set the Menu Content drop-down to (Headings)

Try it! Inserting a Menu Proxy

Create a new MasterPage by right clicking Content Explorer > Resources > MasterPages and selecting New > MasterPage. Once the new MasterPage is in place, click Insert > Proxy > Insert Menu Proxy in the MasterPage. Note that the menu proxy and body proxy typically require some extra styling to get them to show up side-by-side.
